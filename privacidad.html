Privacidad y protección de datos

Responsable del tratamiento: Luis Oscar Soler Bernal (proyecto personal sin ánimo de lucro)
Contacto: hola@luzfija.es

Esta web no requiere registro. El análisis de facturas PDF se realiza en tu navegador: los archivos no se suben ni se almacenan en servidores del responsable. Al navegar por Internet, determinados proveedores (hosting, proxy y proveedores externos de recursos/servicios) pueden tratar datos técnicos necesarios para operar sus servicios.

Qué datos se tratan

  • Datos técnicos de conexión (por ejemplo, dirección IP, cabeceras HTTP, fecha/hora, información del navegador) tratados por proveedores de infraestructura y por proveedores externos al servir recursos.
  • Datos de uso básicos (por ejemplo, páginas vistas, URL de procedencia/referrer, navegador) si está activa la analítica con GoatCounter.
  • Contenido de comunicaciones si nos escribes por email (tu dirección, asunto y mensaje).
  • Datos en tu dispositivo (preferencias/estado/caché) mediante almacenamiento del navegador (localStorage y caché del Service Worker).

Tratamientos, finalidades y base jurídica

1) Hosting (GitHub Pages)

La web está alojada en GitHub Pages. GitHub puede tratar datos técnicos (por ejemplo, IP y logs) para servir el sitio y protegerlo frente a abuso. El responsable no utiliza esos datos con la finalidad de identificar usuarios ni elaborar perfiles. Más información: política de privacidad de GitHub.

Finalidad: prestación del servicio (servir la web) y seguridad.

Base jurídica: interés legítimo (seguridad) y/o necesidad para prestar el servicio solicitado al acceder al sitio.

2) PVPC (proxy en Vercel hacia la CNMC)

Cuando consultas PVPC, la web usa un proxy desplegado en Vercel para resolver restricciones técnicas (por ejemplo, CORS) y operar la funcionalidad de forma fiable. El proxy consulta endpoints públicos de la CNMC. Vercel puede tratar datos técnicos (por ejemplo, IP y logs) para operar y proteger el servicio. El responsable no utiliza esos datos con la finalidad de identificar usuarios ni elaborar perfiles. Más información: política de privacidad de Vercel.

Finalidad: prestar la funcionalidad solicitada (consulta de PVPC) y seguridad/operación del servicio.

Base jurídica: ejecución de la prestación solicitada e interés legítimo (seguridad).

3) Tipografías (Google Fonts)

Esta web carga tipografías desde Google Fonts. Al hacerlo, Google puede recibir datos técnicos (como IP y cabeceras HTTP) necesarios para servir esos archivos. Más información: política de privacidad de Google.

Finalidad: presentación visual del sitio.

Base jurídica: interés legítimo (presentación/funcionamiento del sitio).

4) Librerías JavaScript desde CDNs externos

Para algunas funciones, la web puede cargar librerías desde redes de distribución de contenido (CDNs). Al servir estos archivos, dichos proveedores pueden recibir datos técnicos (como IP y cabeceras HTTP).

  • UNPKG (unpkg.com). UNPKG indica que se ejecuta sobre la red global de Cloudflare (infraestructura de CDN/edge). Más información: UNPKG y política de privacidad de Cloudflare.
  • jsDelivr (jsdelivr.net). Más información: política de privacidad de jsDelivr y lista de subencargados/subprocesadores.
  • SheetJS CDN (cdn.sheetjs.com). Más información: política de privacidad de SheetJS y sitio del CDN de SheetJS.

Nota: Estos proveedores de CDN pueden utilizar infraestructura distribuida globalmente para servir contenido con baja latencia. Para más información sobre transferencias internacionales, consulta el apartado específico más abajo.

Finalidad: prestar funcionalidades solicitadas (procesamiento local en el navegador).

Base jurídica: ejecución de la prestación solicitada (uso de la funcionalidad).

5) Analítica web (GoatCounter)

Esta web usa GoatCounter para estadísticas básicas (páginas vistas, URL de procedencia, navegador). GoatCounter funciona sin cookies. Para contar visitas repetidas, puede tratar temporalmente datos técnicos como IP y User-Agent para asociarlos a un identificador aleatorio de sesión. No se utiliza para identificar usuarios ni elaborar perfiles. Más información: privacidad de GoatCounter.

Finalidad: medición básica del uso del sitio para mejorarlo.

Base jurídica: interés legítimo (mejora del servicio y seguridad).

6) Procesamiento de facturas (local, en tu navegador)

Los PDFs que selecciones se procesan en tu navegador. No se suben ni se almacenan en servidores del responsable. Si para estas funciones se descargan librerías/recursos, se aplican los apartados de Google Fonts y CDNs indicados arriba.

Finalidad: prestar la funcionalidad solicitada (análisis local).

Base jurídica: ejecución de la prestación solicitada.

7) Almacenamiento en tu dispositivo (localStorage y Service Worker)

Para mejorar la experiencia y el rendimiento, el sitio puede usar almacenamiento del navegador:

  • localStorage: preferencias (p. ej. tema claro/oscuro) y datos de estado/caché (p. ej. últimos cálculos o cachés técnicas) para no repetir trabajo.
  • Service Worker / caché: guardar recursos del sitio para acelerar cargas y permitir un funcionamiento más robusto.

Puedes eliminar estos datos borrando los “datos del sitio” desde la configuración de tu navegador.

Finalidad: preferencias y funcionamiento/rendimiento.

Base jurídica: necesidad para prestar el servicio y/o interés legítimo (mejora de experiencia). En el caso de preferencias elegidas por ti (p. ej., tema), se vincula a tu configuración.

8) Si nos escribes por email

Si contactas a través de hola@luzfija.es, trataremos tu dirección de email y el contenido del mensaje únicamente para atender la consulta y responderte.

Finalidad: gestión de consultas.

Base jurídica: interés legítimo (atender comunicaciones y soporte).

Destinatarios

En función del uso del sitio, pueden intervenir los siguientes proveedores:

  • GitHub (hosting mediante GitHub Pages).
  • Vercel (infraestructura del proxy para PVPC).
  • Google (Google Fonts).
  • UNPKG (servicio de CDN) e infraestructura asociada para servir recursos (Cloudflare).
  • jsDelivr (servicio de CDN) y su cadena de subprocesadores según su documentación.
  • SheetJS (servicio de CDN).
  • GoatCounter (estadísticas básicas), si está activo.

Transferencias internacionales

Algunos proveedores pueden estar ubicados fuera del Espacio Económico Europeo o usar infraestructura global. En esos casos, se aplican las condiciones descritas en las políticas de privacidad de cada proveedor.

Conservación

  • Datos tratados por terceros: se conservan según las políticas y plazos de cada proveedor (GitHub, Vercel, Google, Cloudflare/UNPKG, jsDelivr, SheetJS, GoatCounter).
  • Datos en tu dispositivo: permanecen hasta que los borres desde tu navegador (localStorage/caché).
  • Emails: se conservan el tiempo necesario para gestionar la consulta y, posteriormente, durante un plazo razonable para seguimiento o seguridad, salvo que solicites su supresión cuando sea aplicable.

Cookies

Esta web no utiliza cookies de seguimiento ni cookies analíticas basadas en cookies. GoatCounter funciona sin cookies. (El sitio puede usar almacenamiento del navegador como localStorage o caché del Service Worker, explicado arriba.)

Tus derechos

Puedes solicitar el ejercicio de tus derechos de acceso, rectificación, supresión, limitación, portabilidad y oposición escribiendo a hola@luzfija.es.

Si consideras que tus derechos no han sido atendidos adecuadamente, puedes presentar una reclamación ante la Agencia Española de Protección de Datos (AEPD): www.aepd.es.
