Privacidad y protección de datos

Responsable del tratamiento: Luis Oscar Soler Bernal (proyecto personal sin ánimo de lucro)
Contacto: hola@luzfija.es

Esta web no requiere registro, no tiene formularios y no sube tus facturas: el análisis de PDFs se hace en tu navegador. Aun así, al navegar por Internet se tratan datos técnicos (como la dirección IP) por parte del proveedor de hosting y de algunos proveedores externos que sirven recursos (fuentes/librerías) o prestan servicios auxiliares.

Qué datos se tratan y para qué

1) Hosting (GitHub Pages)

La web está alojada en GitHub Pages. Al acceder, GitHub puede tratar datos técnicos de conexión y seguridad (por ejemplo, dirección IP, fecha/hora, cabeceras HTTP) para servir el sitio y protegerlo frente a abuso. Yo no utilizo estos datos para identificarte ni elaboro perfiles. Puedes consultar la información del proveedor aquí: Política de privacidad de GitHub.

Finalidad: prestación del servicio (servir la web) y seguridad.

Base jurídica: interés legítimo y/o necesidad para prestar el servicio solicitado al acceder al sitio.

2) Consulta de PVPC (proxy en Vercel)

Cuando consultas precios PVPC, la petición puede pasar por un proxy desplegado en Vercel para poder realizar la consulta técnica sin exponer credenciales/limitaciones del proveedor de datos y para resolver restricciones de CORS. En ese proceso, Vercel puede tratar datos técnicos (por ejemplo, IP y logs) para operar y proteger el servicio. Yo no uso estos datos para identificar usuarios ni elaborar perfiles. Más información: Política de privacidad de Vercel.

Finalidad: prestar la funcionalidad solicitada (consulta de PVPC) y seguridad/operación del servicio.

Base jurídica: ejecución de la prestación solicitada (uso de la funcionalidad) e interés legítimo (seguridad).

3) Tipografías (Google Fonts)

Esta web carga tipografías desde Google Fonts. Al hacerlo, Google puede recibir datos técnicos (como la IP y cabeceras HTTP) necesarios para servir esos archivos. Más información: Política de privacidad de Google.

Finalidad: presentación visual del sitio.

Base jurídica: interés legítimo (presentación/funcionamiento del sitio).

4) Librerías JavaScript desde CDNs externos

Para algunas funciones (por ejemplo, lectura/análisis de PDFs o importación de ficheros), la web puede cargar librerías desde redes de distribución de contenido (CDNs) como unpkg.com, jsdelivr.net y cdn.sheetjs.com. Estos proveedores pueden recibir datos técnicos (como la IP y cabeceras HTTP) al servir los ficheros.

Finalidad: prestar funcionalidades solicitadas (procesamiento local en el navegador).

Base jurídica: ejecución de la prestación solicitada (uso de la funcionalidad).

5) Analítica web (GoatCounter)

Esta web usa GoatCounter para estadísticas básicas de uso (páginas vistas, referrer, navegador). GoatCounter funciona sin cookies y, para contar visitas repetidas, mantiene temporalmente en memoria (hasta 8 horas) datos técnicos como IP y User-Agent para asociarlos a un identificador aleatorio de sesión. No se utiliza para identificar usuarios ni elaborar perfiles. Más información: Privacidad de GoatCounter.

Finalidad: medición básica del uso del sitio para mejorarlo.

Base jurídica: interés legítimo (mejora del servicio y prevención de abuso).

6) Procesamiento de facturas (local, en tu navegador)

Los PDFs que selecciones se procesan en tu navegador. No se suben a ningún servidor ni se almacenan en bases de datos del sitio. Si usas OCR u otras funciones, se ejecutan igualmente en el navegador (salvo que tu propio navegador contacte con terceros para descargar librerías/recursos, como se indica arriba).

7) Almacenamiento local en tu dispositivo (localStorage y Service Worker)

Para mejorar la experiencia y el rendimiento, el sitio puede usar almacenamiento del navegador:

  • localStorage: preferencias como el tema (claro/oscuro) y, si está habilitado, el último cálculo/estado para no tener que reintroducir datos.
  • Service Worker / caché: guardar recursos del sitio para acelerar cargas y permitir funcionamiento más robusto (por ejemplo, si hay cortes de red).

Puedes eliminar estos datos borrando los "datos del sitio" desde la configuración de tu navegador.

Finalidad: preferencias y funcionamiento/rendimiento.

Base jurídica: necesidad para prestar el servicio y/o interés legítimo (mejora de experiencia). En el caso de preferencias elegidas por ti (p. ej., tema), se consideran vinculadas a tu elección de configuración.

Destinatarios y terceros

Dependiendo de las funciones que uses, pueden intervenir estos proveedores:

  • GitHub (hosting mediante GitHub Pages).
  • Vercel (infraestructura del proxy para PVPC).
  • Google (Google Fonts).
  • CDNs externos (por ejemplo, unpkg.com, jsdelivr.net, cdn.sheetjs.com).
  • GoatCounter (estadísticas básicas).

Transferencias internacionales

Algunos de los proveedores anteriores pueden estar ubicados fuera del Espacio Económico Europeo o usar infraestructura global. En esos casos, se aplican las garantías y condiciones descritas en las políticas de privacidad de cada proveedor.

Conservación

  • Datos tratados por terceros: se conservan según las políticas y plazos de cada proveedor (GitHub, Vercel, Google, CDNs, GoatCounter).
  • Datos en tu dispositivo: permanecen hasta que los borres desde tu navegador (localStorage/caché).

Cookies

Esta web no utiliza cookies de seguimiento ni cookies analíticas basadas en cookies. GoatCounter funciona sin cookies. (El sitio puede usar almacenamiento del navegador como localStorage o caché del Service Worker, explicado arriba.)

Tus derechos

Puedes solicitar el ejercicio de tus derechos de acceso, rectificación, supresión, limitación, portabilidad y oposición escribiendo a hola@luzfija.es.

Si consideras que tus derechos no han sido atendidos adecuadamente, puedes presentar una reclamación ante la Agencia Española de Protección de Datos (AEPD): www.aepd.es.
